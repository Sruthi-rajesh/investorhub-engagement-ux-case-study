# InvestorHub Engagement and UX Case Study

Analytics and UX case study that uses normalized engagement metrics to benchmark companies and propose prioritized UX and communication improvements.

## Deliverables
- **Presentation (PPTX):** `docs/InvestorHub_CaseStudy_SruthiRajesh.pptx`

## Project Overview
**Goal:** Evaluate investor engagement across email + platform usage, identify low-engagement segments, and propose UX + comms improvements.
**Approach:** Normalized metrics (e.g., views per shareholder; estimated email opens) to compare companies fairly, then synthesized findings into prioritized recommendations.
**Output:** Slide deck with analysis, insights, and recommended improvements.

## What I did
- Designed normalized engagement metrics (**views per shareholder**, **estimated email opens**)
- Benchmarked companies to identify low-engagement segments
- Proposed UX and communication improvements tied to findings

## Key Highlights
- Built a normalized engagement framework across channels
- Identified the lowest-engagement company/segment using comparable metrics
- Proposed UX and communication changes based on evidence from the analysis
